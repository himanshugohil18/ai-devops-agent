# AI DevOps Log Analyzer Agent

An AI-powered DevOps agent that analyzes logs, detects errors, identifies root causes,
and suggests fixes using rule-based logic and LLMs.

## Features
- Log cleaning & preprocessing
- Rule-based known error detection
- AI-powered root cause analysis
- CLI based MVP

## Run
1. Add OpenAI key in `.env`
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   python main.py
